## Overview
This is a single-page portfolio website built with **HTML, CSS, and JavaScript**. It demonstrates:

- A clean and responsive design
- Interactive animations for a professional look
- Structured sections for About, Skills, Projects, Blogs, Certifications, Contact, and Login

The design adapts seamlessly to desktop, tablet, and mobile devices.

---

## Features
- **Responsive Header** with a hamburger menu for mobile devices
- **About Me Section** with image and detailed introduction
- **Skills Section** with professional skill cards including HTML, CSS, JavaScript, React, Node.js, Angular, MongoDB, MySQL
- **Projects/Cards Section** highlighting different roles like Frontend, Backend, Full Stack, and UI/UX Designer
- **Blogs Section** for showcasing achievements or posts
- **Certifications Section** with animated certification cards
- **Contact Form** for user interaction
- **Login Form** with clean UI
- **Professional Footer** with social links and contact info
- **Smooth Animations** (fade-in left, right, top, bottom)
